# Modern PDF Toolkit

Feature list:
- Merge PDF
- Split PDF
- Rotate PDF
- Page numbers
- Watermark (text/image)
- Protect PDF
- Unlock PDF (known password)
- Organize PDF (reorder/delete pages)
- Image to PDF
- PDF to image
- Crop PDF
- Compress PDF (30/50/100/200/300/500 KB + manual size)

## Run

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

Open: http://localhost:5000
